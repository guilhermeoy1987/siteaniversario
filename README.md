# 🎂 Projeto Landing Page de Aniversário

Este repositório contém o desenvolvimento de um site de evento personalizado, focado em design moderno e interatividade para celebrações.

## O que foi realizado:
* **UI/UX Design:** Criação de uma interface visualmente atraente e fácil de navegar.
* **Componentes Dinâmicos:** Uso de scripts para melhorar a interação do usuário com a página.
* **Estilização com SASS:** Organização de estilos utilizando variáveis e aninhamento para facilitar a manutenção.
* **Mobile First:** Layout totalmente responsivo, garantindo acessibilidade em todos os tamanhos de tela.

## Tecnologias:
* HTML5
* CSS3 / SASS
* JavaScript
* Git / GitHub
